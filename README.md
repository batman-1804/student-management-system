# Student Management System

## Objective
The objective of this project is to manage student records efficiently using a simple Python-based application.

## Description
This system allows the user to add, view, search, and delete student records. The data is stored in a text file for simplicity.

## Technologies Used
- Python
- File Handling

## Features
- Add new student details
- View all student records
- Search student by roll number
- Delete student record

## How to Run
1. Install Python
2. Download or clone the repository
3. Run the file using:
   python main.py
